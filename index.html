形狀消消樂網頁版 Shape Clear - HTML

形狀消消樂網頁版是一個消消樂遊戲，使用 Phaser 遊戲引擎和 HTML5 規格

目前還沒有完成，但是可以玩😆~~

玩法

用滑鼠或手指交換 2 個形狀，讓 3 個一樣的形狀，連成一條線，就可以消除

馬上玩！

本遊戲支援離線遊玩，如果覺得遊戲沒有更新，可以按這裡強制更新

無法玩遊戲？

如果遊戲畫面一直閃，畫面一團亂，或者畫面一片空白，這可能是因為您的裝置不支援硬體加速

你可以進入設定，將 renderer 改成 CANVAS

如果你只看到「Unable to load Phaser」，那我猜你用了非常舊的瀏覽器。 我的網站不支援 IE 9 以下的版本，請改用 Chrome 或 Firefox

如果卡在「Loading assets...」，有可能是

  1. 網絡不給力
  2. 你的瀏覽器不能解碼聲音

無論是哪種原因，你都可以進入設定，將 Disable audio 打勾

遊戲有bug！

還是一樣，你可以到我的程式庫上面回報問題，就在 github.com/stdio2016/shapeclear-html/issues。 或者你比較喜歡 Facebook，就私訊我的粉絲專頁吧
